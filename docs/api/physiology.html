# Physiology

# computeGlicemia

function [G, insulinBolus, correctionBolus, insulinBasal, CHO, hypotreatments, x] = computeGlicemia(mP,data,model,dss)

Compute the glycemic profile obtained with the ReplayBG physiological model using as integration step 1 minute and using the given data and model parameters.

# Inputs

  • mP: structure (required)
    A struct containing the model parameters;
  • data: timetable (required)
    A timetable which contains the data to be used by the tool;
  • model: structure (required)
    A structure that contains general parameters of the physiological model;
  • dss: structure (required)
    A structure that contains the hyperparameters of the integrated decision support system.

# Outputs

  • G: vector of double
    A vector containing the simulated glucose trace (mg/dl);
  • insulinBolus: vector of double
    A vector containing the bolus insulin used to obtain G (U/min);
  • correctionBolus: vector of double
    A vector containing the correction bolus insulin used to obtain G (U/min);
  • insulinBasal: vector of double
    A vector containing the basal insulin used to obtain G (U/min);
  • CHO: vector of double
    A vector containing the CHO used to obtain glucose (g/min);
  • hypotreatments: vector of double
    A structure containing the hypotreatments used to obtain G (g/min);
  • x: matrix of double
    A matrix containing the simulated model state trajectories.

# modelStep

function xk = modelStep(xkm1,B,CHO,mP,xk,model)

Simulates a step of the physiological model using the Backward Euler's algorithm.

# Inputs

  • xkm1: vector of double (required)
    A vector containing the model state at time k-1;
  • B: double (required)
    The insulin at time k (mU/min*kg);
  • CHO: double (required)
    The CHO at time k (mg/min*kg);
  • mP: structure (required)
    A struct containing the model parameters;
  • xk: vector of double (required)
    The preallocated vector that will contain the model state at time k;
  • model: structure (required)
    A structure that contains general parameters of the physiological model.

# Output

  • xk: vector of double
    A vector containing the model state at time k.

# insulinSetup

function [bolus, basal] = insulinSetup(data,model,modelParameters)

Generates the vector containing the insulin data to be used to simulate the physiological model.

# Inputs

  • data: timetable (required)
    A timetable which contains the data to be used by the tool;
  • model: structure (required)
    A structure that contains general parameters of the physiological model;
  • modelParameters: structure (required)
    A structure containing the model parameters.

# Outputs

  • bolus: vector of double
    A vector containing the insulin bolus dose at each time step (mU/min*kg);
  • basal: vector of double
    A vector containing the basal insulin value at each time step (mU/min*kg).

# mealSetup

function [meal] = mealSetup(data,model,modelParameters)

Generates the vector containing the CHO intake events to be used to simulate the physiological model.

# Inputs

  • data: timetable (required)
    A timetable which contains the data to be used by the tool;
  • model: structure (required)
    A structure that contains general parameters of the physiological model;
  • modelParameters: structure (required)
    A structure containing the model parameters.

# Outputs:

  • meal: vector of double (required)
    A vector containing the carbohydrate intake at each time step (mg/min*kg).
