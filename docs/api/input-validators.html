# ReplayBG Input Validators

# modalityValidator

function  modalityValidator(modality)

Validates the input parameter modality.

# Input

  • modality: vector of char, {'identification', 'replay'} (required)
    Specifies if the function will be used to identify the ReplayBG model on the given data or to replay the scenario specified by the given data.

# Output

  • valid: boolean
    A boolean defining if the input parameter is valid.

# MCMCTheta0Validator

function valid = MCMCTheta0PolicyValidator(MCMCTheta0Policy,modality)

Validates the input parameter MCMCTheta0Policy.

# Inputs

  • MCMCTheta0Policy: vector of char, {'mean', 'initial', 'last'}, default: 'mean'
    A vector of characters defining the policy used by the MCMC procedure to set the initial MCMC chain values. Using 'mean', the mean value of the MCMC chain obtained from the last MCMC run will be set as initial MCMC chain value to be used in the next MCMC run. Ignored when modality is 'replay';
  • modality: vector of char, {'identification', 'replay'} (required)
    Specifies if the function will be used to identify the ReplayBG model on the given data or to replay the scenario specified by the given data.

# Output

  • valid: boolean
    A boolean defining if the input parameter is valid.

# bayesianEstimatorValidator

function valid = bayesianEstimatorValidator(bayesianEstimator,modality)

Validates the input parameter bayesianEstimator.

# Inputs

  • bayesianEstimator: vector of char, {'mean', 'map'}, default: 'mean'
    A vector of characters defining which Bayesian estimator to use to obtain a point estimate of model parameters. Using 'mean' the posterior mean estimater will be used. Ignored when modality is 'replay';
  • modality: vector of char, {'identification', 'replay'} (required)
    Specifies if the function will be used to identify the ReplayBG model on the given data or to replay the scenario specified by the given data.

# Output

  • valid: boolean
    A boolean defining if the input parameter is valid.

# preFilterDataValidator

function valid = preFilterDataValidator(preFilterData,modality)

Validates the input parameter preFilterData.

# Inputs

  • preFilterData: integer, {0, 1}, default: 0
    A numerical flag that specifies whether to filter the glucose data before performing the model identification or not. This might help the identification procedure. Ignored when modality is 'replay';
  • modality: vector of char, {'identification', 'replay'} (required)
    Specifies if the function will be used to identify the ReplayBG model on the given data or to replay the scenario specified by the given data.

# Output

  • valid: boolean
    A boolean defining if the input parameter is valid.
