# Visualization

function plotReplayBGResults(glucose,data,environment)

Plot the obtained glucose results against data.

# Inputs

  • glucose: structure (required)
    A structure which contains the obtained glucose traces simulated via ReplayBG;
  • data: timetable (required)
    Timetable which contains the data to be used by ReplayBG;
  • environment: timetable (required)
    A structure that contains general parameters to be used by ReplayBG.
