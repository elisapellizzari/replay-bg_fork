# Analysis

# analyzeResults

function analysis = analyzeResults(glucose, insulinBolus, correctionBolus, insulinBasal, CHO, hypotreatments,data,environment)

Analyses the simulated 1000 glucose traces obtained using ReplayBG.

# Inputs

  • glucose: structure (required)
    A structure which contains the obtained 1000 glucose traces simulated via ReplayBG
  • insulinBolus: structure (required)
    A structure containing the input bolus insulin used to obtain glucose (U/min);
  • correctionBolus: structure (required)
    A structure containing the correction bolus insulin used to obtain glucose (U/min);
  • insulinBasal: structure (required)
    A structure containing the input basal insulin used to obtain glucose (U/min);
  • CHO: structure (required)
    A structure containing the input CHO used to obtain glucose (g/min);
  • hypotreatments: structure (required)
    A structure containing the input hypotreatments used to obtain glucose (g/min).
  • data: timetable (required)
    A timetable which contains the data to be used by the tool;
  • environment: structure (required)
    A structure that contains general parameters to be used by ReplayBG.

# Output

  • analysis: structure
    A structure that contains the results of the analysis of the simulated glucose traces obtained using ReplayBG.

# replayScenario

function [glucose, insulinBolus, correctionBolus, insulinBasal, CHO, hypotreatments] = replayScenario(data,modelParameters,draws,environment,model,mcmc,dss)

Replays the scenario defined by the given data and provides 1000 realizations of glucose (one for each draw).

# Inputs

  • data: timetable (required)
    A timetable which contains the data to be used by the tool;
  • modelParameters: structure (required)
    A struct containing the model parameters;
  • draws: structure (required)
    A structure that contains 1000 realizations of modelParameter obtained with MCMC;
  • environment: structure (required)
    A structure that contains general parameters to be used by ReplayBG;
  • model: structure (required)
    A structure that contains general parameters of the physiological model;
  • mcmc: structure (required)
    A structure that contains the hyperparameters of the MCMC identification procedure;
  • dss: structure (required)
    A structure that contains the hyperparameters of the integrated decision support system.

# Outputs

  • glucose: structure
    A structure which contains the obtained 1000 glucose traces simulated via ReplayBG
  • insulinBolus: structure
    A structure containing the input bolus insulin used to obtain glucose (U/min);
  • correctionBolus: structure
    A structure containing the correction bolus insulin used to obtain glucose (U/min);
  • insulinBasal: structure
    A structure containing the input basal insulin used to obtain glucose (U/min);
  • CHO: structure
    A structure containing the input CHO used to obtain glucose (g/min);
  • hypotreatments: structure
    A structure containing the input hypotreatments used to obtain glucose (g/min).
