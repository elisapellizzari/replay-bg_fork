# Default Decision Support Handlers

# correctsAbove250Handler

function CB = correctsAbove250Handler(G,CHO,bolus,basal,time,timeIndex)

Implements the default correction bolus strategy: "take a correction bolus of 1 U every 1 hour while above 250 mg/dl".

# Inputs

  • G: double (required)
    The glucose concentration at time(timeIndex)
  • CHO: vector of double (required)
    A vector that contains the CHO intakes input for the whole replay simulation (g/min);
  • bolus:vector of double (required)
    A vector that contains the bolus insulin input for the whole replay simulation (U/min);
  • basal: vector of double (required)
    A vector that contains the basal insulin input for the whole replay simulation (U/min);
  • time: vector of datetime (required)
    A vector that contains the time istants of current replay simulation. Contains one value for each integration step;
  • timeIndex: integer (required)
    A number that defines the current time istant in the replay simulation.

# Output

  • CB: double
    The correction bolus to administer at time(timeIndex+1) (U/min).

# adaHypoTreatmentsHandler

function HT = adaHypoTreatmentsHandler(G,CHO,bolus,basal,time,timeIndex)

Implements the default hypotreatment strategy: "take an hypotreatment of 10 g every 15 minutes while in hypoglycemia".

# Inputs

  • G: double (required)
    The glucose concentration at time(timeIndex)
  • CHO: vector of double (required)
    A vector that contains the CHO intakes input for the whole replay simulation (g/min);
  • bolus:vector of double (required)
    A vector that contains the bolus insulin input for the whole replay simulation (U/min);
  • basal: vector of double (required)
    A vector that contains the basal insulin input for the whole replay simulation (U/min);
  • time: vector of datetime (required)
    A vector that contains the time istants of current replay simulation. Contains one value for each integration step;
  • timeIndex: integer (required)
    A number that defines the current time istant in the replay simulation.

# Output

  • HT: double The hypotreatment to administer at time(timeIndex+1) (g/min).

# Note on the function handlers

  • Integration step is 1 minute. Function handlers of the integrated decision support systems works at this time step. So, input vectors of the function handlers contain one value for each integration step.
