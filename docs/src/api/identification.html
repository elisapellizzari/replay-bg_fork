# Identification

# setModelParameters

function [modelParameters, mcmc, draws] = setModelParameters(data,BW,environment,mcmc,model)

Sets the parameters of the physiological model.

# Inputs:

  • data: timetable (required)
    A timetable which contains the data to be used by the tool;
  • BW: double (required)
    The patient's body weight;
  • environment: structure (required)
    A structure that contains general parameters to be used by ReplayBG;
  • mcmc: structure (required)
    A structure that contains the hyperparameters of the MCMC identification procedure;
  • model: structure (required)
    A structure that contains general parameters of the physiological model;
  • dss: structure (required)
    A structure that contains the hyperparameters of the integrated decision support system.

# Outputs:

  • modelParameters: structure
    A struct containing the model parameters;
  • mcmc: structure
    The updated structure that contains the hyperparameters of the MCMC identification procedure;
  • draws: structure
    A structure that contains 1000 realizations of modelParameter obtained with MCMC.

# identifyModelParameters

function [modelParameters, draws] = identifyModelParameters(data, BW, mcmc, model, dss, environment)

Identifies the physiological model parameters using MCMC.

# Inputs

  • data: timetable (required)
    A timetable which contains the data to be used by the tool;
  • BW: double (required)
    The patient's body weight;
  • mcmc: structure (required)
    A structure that contains the hyperparameters of the MCMC identification procedure;
  • model: structure (required)
    A structure that contains general parameters of the physiological model;
  • dss: structure (required)
    A structure that contains the hyperparameters of the integrated decision support system;
  • environment: structure (required)
    A structure that contains general parameters to be used by ReplayBG.

# Outputs

  • modelParameters: structure
    A struct containing all the identified model parameters obtained using the specified bayesian estimator;
  • draws: structure
    A structure that contains 1000 realizations of modelParameter obtained with MCMC.

# runMCMC

function [pHat, accept, ll] = runMCMC(data,mcmc,mP,model,dss,environment)

Performs a run of the MCMC identification procedure.

# Inputs

  • data: timetable (required)
    A timetable which contains the data to be used by the tool;
  • mcmc: structure (required)
    A structure that contains the hyperparameters of the MCMC identification procedure;
  • mP: structure (required)
    A struct containing all the identified model parameters obtained using the specified bayesian estimator;
  • model: structure (required)
    A structure that contains general parameters of the physiological model;
  • dss: structure (required)
    A structure that contains the hyperparameters of the integrated decision support system;
  • environment: structure (required)
    A structure that contains general parameters to be used by ReplayBG.

# Outputs

  • pHat: structure
    A structure containing the MCMC chain realizations;
  • accept: vector of integer
    A vector containing the acceptance rate of each MCMC block;
  • ll: vector of double
    A vector containing the values the likelihood take through the simulation.

# setNewMCMCParameters

function  [mcmc] = setNewMCMCParameters(pHat,mcmc)

Computes the conditional standard deviation for all the estimated parameters (as described in MCMC in practice, Gilks, pp. 123) and sets the new values of mcmc.std and mcmc.theta0.

# Inputs

  • pHat: structure (required)
    A structure containing the MCMC chain realizations;
  • mcmc: structure (required)
    A structure that contains the hyperparameters of the MCMC identification procedure.

# Output

  • mcmc: structure
    The updated structure that contains the hyperparameters of the MCMC identification procedure.

# reftery_lewis

function  [raftery_lewis] = raftery_lewis(runs,q,r,s)

Computes the convergence diagnostics of Raftery and Lewis (1992), i.e. the number of draws needed in MCMC to estimate the posterior cdf of the q-quantile within an accuracy r with probability s

# Inputs

  • draws: matrix of double (required)
    A double matrix of draws from the MCMC;
  • q: double (required)
    The quantile of the quantity of interest;
  • r: double (required)
    The level of desired precision;
  • s: double (required)
    The probability associated with r.

# Output

  • raftery_lewis: structure
    A structure that contains the results of the convergence analysis.
